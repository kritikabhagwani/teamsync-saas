# TeamSync

A SaaS Project Management application where organizations can create projects, assign tasks, chat with teammates, track progress, and collaborate in real time.

## Features (Planned)

- User Authentication
- Organizations
- Team Management
- Project Management
- Kanban Board
- Task Assignment
- Real-time Chat
- Notifications
- Dashboard & Analytics
- AI-powered Task Assistance

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express

Database
- MongoDB

Real-time
- Socket.IO

Authentication
- JWT

Storage
- Cloudinary
